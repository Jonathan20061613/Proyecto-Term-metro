TERMÓMETRO DE TEMPERATURA Y HUMEDAD

¡Bienvenido a la documentación de nuestro innovador proyecto de monitoreo ambiental! Este sistema utiliza un Arduino UNO, un sensor DHT22 y una pantalla LCD con I2C para medir y mostrar en tiempo real los niveles de temperatura y humedad del ambiente.

Diseñado especialmente para aplicaciones en invernaderos y gallineros, este proyecto tiene como objetivo asegurar condiciones óptimas que promuevan el bienestar de las gallinas y optimicen la producción de huevos. A través de un diseño accesible y funcional, los cuidadores pueden supervisar el ambiente de manera eficiente, tomando decisiones informadas para garantizar un entorno saludable y productivo.

Explora esta documentación para conocer más sobre los componentes utilizados, la funcionalidad del sistema y los resultados esperados. ¡Gracias por unirte a nosotros en este viaje hacia un mejor manejo ambiental!

Acerca del Proyecto

Este proyecto tiene como objetivo desarrollar un termómetro de temperatura y humedad utilizando tecnología de Arduino, específicamente un Arduino UNO, un sensor DHT22 y una pantalla LCD con I2C. La idea es crear un sistema accesible y eficiente que permita medir y mostrar en tiempo real los niveles de temperatura y humedad en diversos entornos, como invernaderos y gallineros.

El sensor DHT22 proporciona mediciones precisas, que se visualizan de manera clara y concisa en la pantalla LCD, facilitando el monitoreo continuo de las condiciones ambientales. Esto es crucial para asegurar un ambiente óptimo para el bienestar de las gallinas, influyendo positivamente en su salud y producción de huevos.

Además, el sistema está diseñado para ser escalable, permitiendo la integración de alertas visuales y sonoras en caso de que las condiciones superen rangos críticos. De esta manera, los cuidadores pueden tomar decisiones informadas y realizar ajustes en el entorno de forma proactiva.

Este proyecto no solo destaca la importancia del uso de tecnología en la agricultura y la cría de animales, sino que también sirve como una excelente oportunidad de aprendizaje práctico en el ámbito de la electrónica y la programación.

Componentes Utilizados

  • Arduino UNO
  • Sensor DHT22
  • Pantalla LCD con I2C
  • Cables de conexión

Resultados Esperados
